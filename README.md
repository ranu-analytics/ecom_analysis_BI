## E-Commerce Sales Dashboard 


![E-Commerce Dashboard](ecom-dashboard.png)
---

###  Project Overview  
This project presents an interactive E-Commerce Sales Dashboard built using Power BI. It provides insights into product performance, sales trends, and customer purchasing behavior.

---

### Key Metrics  
-  Total Sales: 2B  
-  Total Quantity: 47K  
-  Orders: 525M  
-  Total Price: 3M  

---

###  Insights  
-  Price distribution by product  
-  Category-wise sales performance  
-  Sales analysis by city  
-  Product demand trends  
-  High-performing products  

---

### Key Highlights  
- Top-selling category: **Laptop & Accessories**  
- Strong sales contribution from major cities  
- Wide variation in product pricing  
- Consistent demand across categories  

---

###  Tools Used  
- Power BI  
- DAX  
- Data Visualization
